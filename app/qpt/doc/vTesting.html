QPT Testing Release Notes

This is a testing version of QPT. This addresses 2 issues, described below.

The QPT documentation index is available here.

To install and run the test release of QPT:
Remove previous test version → launcher -a:qpt-test -u
To install → launcher -i:qpt -a:qpt-test -h:build.cl.gemini.edu -p:9997
To run → launcher -a:qpt-test
To uninstall → launcher -a:qpt-test -u
As always, the first time you run QPT it will download necessary components. This may take several minutes, especially in the north. Your previous installation of QPT should continue to function normally.
