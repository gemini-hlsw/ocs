ITC Wishlist

The wishlist of modifications and improvements for possible inclusion in future releases is:

  • Enable ITC calculation for observations defined in Phase II Observing Tool, using instrument configuration and other components to set parameters, using observation sequence to guess observation method. User still needs to specify source brightness and morphology.
  • Re-bin output spectra to arbitrary user-defined resolution (default to be nominal resolution); note: non-trivial to correctly propagate noise
